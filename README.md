# Mafia Flip (f.t. Miboard flip 2)

Have you ever wantee to use your keyboard as a mouse? No? it's because you're weird. Normal ppl would.

It uses vaydeer keypad with an esp32 and a gyroscope.

## Requirements

- esp-idf software download here: https://docs.espressif.com/projects/esp-idf/en/stable/esp32/get-started/index.html
- Godot
- python with serial installed
- Twillo (with free trial)

## How to use?

- flip the miboard to get the joystick (The gyroscope recognized the x axis angle change, and assigns it a value either UP or down)
- flip it back to has a 9 key keyboard to use to flee the cops
